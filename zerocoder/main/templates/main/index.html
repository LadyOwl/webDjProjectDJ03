Добро пожаловать на сайт {{caption}}!

Этот проект создан в рамках изучения фреймворка Django — мощного инструмента для разработки веб-приложений на Python.

Здесь я экспериментирую с шаблонами, статическими файлами, маршрутами и базами данных.

Сайт построен с использованием Bootstrap 5, чтобы выглядел современно и корректно отображался на всех устройствах.

На данный момент реализованы две страницы: главная и второстепенная.

В будущем планируется добавить:

  • Форму обратной связи
  • Систему пользователей
  • Динамический контент

Django позволяет быстро и безопасно создавать сложные веб-приложения, не изобретая велосипед.

Все статические файлы (CSS, изображения) организованы по рекомендациям документации.

Проект разрабатывается локально, но в перспективе может быть развёрнут на реальном сервере.

Особое внимание уделено чистоте кода и правильной архитектуре приложения.

Логотип «Rainbow» символизирует разнообразие возможностей веб-разработки.
